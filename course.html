Noob to Pro Syllabus

Progress step-by-step from fundamentals to confident coding.

Unit 1 – Getting Started (Noob Stage)

  • Install Go, understand workspace, and run first program.
  • Understand `package main` and project structure.
  • Assessment: Quiz + Big Test included.

Unit 2 – Variables & Data Types (Learner Stage)

  • Work with variables, constants, and core data types.
  • Perform type conversion and format outputs cleanly.
  • Assessment: Quiz + Big Test included.

Unit 3 – If & Switch (Builder Stage)

  • Build decision logic using if/else and switch.
  • Solve practical branching exercises.
  • Assessment: Quiz + Big Test included.

Unit 4 – Loops (Pro Stage)

  • Master loop patterns and control flow for problem solving.
  • Apply concepts in mini projects and coding challenges.
  • Assessment: Quiz + Big Test included.
