# AGENTS Instructions

## Project Scope
- This file applies to the entire repository.

## Tech Stack
- This is a static website built from standalone HTML files.
- Keep pages dependency-free unless explicitly requested.

## Editing Rules
- Preserve the current visual style and simple structure.
- Reuse existing section patterns and class naming conventions.
- Keep HTML semantic and accessible (headings in order, descriptive links, alt text for images).
- Use relative links for internal navigation.

## Validation
- Before finishing, run a quick syntax check by opening files with a validator tool if available, or at minimum ensure changed HTML is well-formed.
- Verify there are no broken internal links introduced by your changes.

## Delivery
- Keep changes focused and minimal.
- Document what changed and which files were touched.
